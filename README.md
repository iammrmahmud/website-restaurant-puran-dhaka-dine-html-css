<h2><a class="anchor" id="overview"></a>Overview</h2>

This project was developed as part of the **CST 2310 - Web Programming 1** midterm project. It is a fully functional **3-page restaurant website** that includes **About Us**, **Contact Us**, and **Reservation** pages.

The website was built using **HTML5** and **CSS3** to demonstrate semantic page structure, consistent styling, responsive design, and deployment workflow. The final project was deployed on **Netlify** as a live website.

---

<h2><a class="anchor" id="problem-statement"></a>Problem Statement</h2>

Many small restaurants do not have a simple and organized online presence where customers can:
- learn about the restaurant
- find contact information
- view location details
- request reservations

This project demonstrates how a front-end website can solve these problems by creating a clean and user-friendly digital presence for a restaurant.

<h2><a class="anchor" id="methods"></a>Methods</h2>

The project was completed by following a structured front-end development process:

- Built a multi-page website using semantic HTML5
- Used a single external CSS stylesheet for consistent design
- Created responsive layouts with media queries
- Added navigation links across all pages
- Built and styled contact and reservation forms
- Embedded a location map using iframe
- Organized files into separate folders for better maintainability
- Deployed the final project using Netlify
